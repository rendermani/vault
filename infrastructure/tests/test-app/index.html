Testing Instructions

  1. Run docker-compose -f docker-compose.local-test.yml up -d
  2. Wait for all services to become healthy (check status above)
  3. Test service connectivity using the dashboard links
  4. Run deployment script: ./scripts/unified-bootstrap.sh --environment develop --dry-run
  5. Clean up: docker-compose -f docker-compose.local-test.yml down -v

Remote Deployment

Once local testing passes, deploy to remote server using GitHub Actions:

  • Push changes to trigger automatic deployment
  • Or use manual workflow dispatch for controlled deployment
  • Monitor deployment progress in GitHub Actions
